Key Features

STABLECOIN

Fiat-Backed Stablecoin

VCS token with 1:1 USD peg and full reserve backing. Request-based minting with approval workflow and reserved burn system.

SECURITY

Enterprise Security

Proof of Authority consensus with trusted validators, multi-signature support, and emergency pause mechanism for critical operations.

KYC

KYC Integration

Built-in identity verification with document hash submission, multi-tier workflow, and KYC verifier role management.

COMPLIANCE

Compliance Monitoring

Account flagging system, suspicious activity alerts, transaction limit monitoring, and AML integration ready.

REST API

REST API Gateway

Complete backend with JWT authentication, wallet operations, payment processing, and admin dashboard. 2,600+ lines of production code.

WINDOWS

Windows Compatible

Production-ready batch scripts for multi-validator setup with persistent node keys. Pure CMD, no PowerShell required.

Quick Start

1

Build Blockchain

Clone the repository and build the Substrate node:

cd velo-chain
rustup target add wasm32-unknown-unknown
cargo build --release
2

Setup Database

Create PostgreSQL database and run migrations:

createdb velopay
cd velopay-api
sqlx migrate run
3

Start Services

Launch the blockchain and API gateway:

# Terminal 1: Blockchain
./target/release/velo-node --dev

# Terminal 2: API
cd velopay-api
cargo run --release

Windows Multi-Validator Setup

cd velo-chain
setup-node-keys-v2.bat
run-alice-final.bat      REM Terminal 1
run-bob-final.bat        REM Terminal 2
